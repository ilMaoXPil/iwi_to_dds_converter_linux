# iwi_to_dds_converter_linux
Desktop Shortcut that allows Linux/Steam Deck users to drag and drop custom dds files for Classic Call Of Duty Games. (Original App by kokole)

# INSTRUCTIONS:

1: Download the latest version of Wine (For Steam Deck/Arch users you open the terminal and type "sudo pacman -S wine")

2: Go to the releases section of the repository and download the zip to a location of your choice

3: Extract the zip

4: Open iwi2dds.desktop with a Text editor of your choice and change the file location "/home/lucas/.local/share/Steam/steamapps/compatdata/3500955683/pfx/drive_c/users/steamuser/AppData/Local/Plutonium/iwi_tools/iwi_dds_fast_converter.exe" to whereever you extracted the zip file to.

5: Don't touch anything else and you should be able to drag and drop to convert files just like on windows.

# (ONLY TESTED ON KDE PLASMA)
